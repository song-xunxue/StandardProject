# StandardProject


- 个人所做的标准化项目
### 项目整体概述
- 本仓库为个人标准化开源项目合集，聚焦C++ 底层开发、前端界面开发、Qt应用开发等方向，涵盖高并发内存管理、Coze智能体应用和SDK 前端落地、音乐播放器开发等实战场景。
- 项目代码规范、结构模块化，兼顾学习性、实用性与工程性，适配面试参考、技术研究、二次开发等多种需求。
- 仓库主开发语言为 C++ 和 Python，辅以 Makefile、Python、HTML/CSS 等技术栈，各项目独立可编译、可运行，具备清晰的开发迭代记录与功能优化轨迹。

### 仓库核心项目列表
- MY_ConcurrentMemoryPool：C++ 高性能高并发内存池，完成基数树优化与多线程测试

- MY_Coze：Coze应用SDK + 前端封装

- MY_MusicPlayer：轻量级音乐播放器项目，完成 CozeSDK 初始化适配

- MY_NLPAnalyzer：自然语言处理分析工具，基于Coze和Langchain
