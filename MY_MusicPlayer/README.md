# MY_MusicPlayer - Qt 轻量级音乐播放器

## 项目简介

MY_MusicPlayer 是基于 **Qt 框架**开发的桌面端本地音乐播放器，采用 **C++ / Qt Widgets** 实现完整的图形化交互与音频播放能力。项目依托 Qt 自带多媒体模块、文件系统与界面组件，实现了本地音乐加载、播放控制、播放列表管理等核心功能，界面简洁、逻辑清晰，是 Qt 桌面开发、音视频编程的经典实战项目。



---

## 技术栈

- 开发语言：C++11+

- 开发框架：Qt 5 / Qt 6（Widgets + Multimedia）

- 核心模块：`QMediaPlayer`、`QMediaPlaylist`、文件系统、UI 布局、信号槽机制

- 构建工具：qmake / Qt Creator 编译运行

- 运行平台：Windows / Linux 跨平台

---

## 功能特性

1. **Qt 图形化界面**

    - 可视化操作界面，按钮、列表、状态展示完整

    - 采用 Qt 标准布局，界面简洁易用

2. **本地音乐文件加载**

    - 支持打开本地音频文件（MP3、WAV、FLAC 等 Qt 支持格式）

    - 自动识别文件名称、路径、格式

3. **完整播放控制**

    - 播放 / 暂停 / 停止

    - 上一曲 / 下一曲

    - 播放进度实时显示（支持进度条）

    - 音量调节

4. **播放列表管理**

    - 多首歌曲添加、展示

    - 列表点击切歌

    - 顺序播放逻辑

5. **Qt 多媒体底层驱动**

    - 基于 `QMediaPlayer` 实现音频解码与播放

    - 基于 `QMediaPlaylist` 管理播放队列

    - 使用 Qt **信号与槽**实现 UI 与播放逻辑解耦

## 项目结构

```Plain Text

MY_MusicPlayer/
├── main.cpp               # 程序入口，初始化主窗口
├── MusicPlayer.h          # 播放器主窗口类定义
├── MusicPlayer.cpp        # 主窗口逻辑实现：UI、信号槽、播放控制
├── MusicPlayer.ui         # Qt Designer 可视化界面文件
└── MusicPlayer.pro        # Qt 项目构建文件
```

### 核心代码说明

- **MusicPlayer 类**：主窗口，整合播放器所有功能

- **UI 交互**：按钮点击、列表选择、进度条拖动

- **QMediaPlayer**：负责音频播放、暂停、音量、进度获取

- **QMediaPlaylist**：管理播放列表、切换歌曲

- **信号槽机制**：实现界面操作与播放状态同步

## 运行方式

项目支持两种运行方式，可根据需求选择，操作简单便捷：

1. **方式一：解压安装包直接运行（推荐，无需编译）**

2. 下载项目发布的安装包（压缩文件）

3. 解压至任意文件夹，无需额外配置

4. 找到解压目录下的可执行文件（.exe 或对应系统可执行程序），双击即可运行

5. **方式二：Qt 编译运行（适合开发、调试）**

6. 使用 **Qt Creator** 打开 `MusicPlayer.pro`

7. 配置 Qt 套件（Desktop Qt 5.12+ / Qt 6.x）

8. 直接编译运行

9. 点击「打开文件」添加音乐，即可播放
